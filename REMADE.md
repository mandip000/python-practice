# Student Info System

This is a beginner Python mini project.

## Features
- Add student name and marks
- Search student details
- Store data in CSV file
- Simple menu system

## Technologies Used
- Python
- CSV file handling

## How to Run

1. Install Python
2. Open project in VS Code
3. Run the file:

python student_info_system.py
